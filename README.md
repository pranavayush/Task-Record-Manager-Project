# 📋 Task Record Manager Project

## 📌 Overview

Task Record Manager is a web-based application designed to efficiently manage, organize, and track daily tasks. The system allows users to create, update, delete, and monitor task records while maintaining productivity and workflow management.

## ✨ Features

### 📝 Task Management

* Add new tasks
* Edit existing tasks
* Delete tasks
* View all task records
* Search and filter tasks

### 📊 Task Tracking

* Track task status
* Mark tasks as Completed, Pending, or In Progress
* View task history
* Monitor productivity

### 👤 User-Friendly Interface

* Responsive design
* Easy navigation
* Clean dashboard
* Fast performance

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/your-username/Task-Record-Manager-Project.git
```

### 2. Navigate to Project Directory

```bash
cd Task-Record-Manager-Project
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Configure Environment Variables

Create a `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
```

### 5. Start the Project

```bash
npm run dev
```
